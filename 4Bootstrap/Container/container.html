What is the Difference between Fluid Container and Container

Although these two classes are very similar, they are slightly different in their width properties. The container class has a fixed width that it can go to based on the screen size. The fluid container class makes the image stay at 100% based on its height and width parameters. This means the image will not be distorted and you can easily change the size of the image. Without parameters, an image with the fluid container class can spread across the full page which is farther than the fixed width of the container class. In conclusion, the fluid container is easier for versatility and change while container is better for full page usage.
